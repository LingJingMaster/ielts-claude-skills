## 算分公式

总分 = 四科平均，四舍五入到 0.5。**.25 和 .75 向上取整**（7.25→7.5，6.75→7.0）。

**常见目标的达成组合：**
- 目标 7.5 = L8 + R8 + W6.5 + S6.5（29 ÷ 4 = 7.25 → 7.5）
- 目标 7.0 = L7.5 + R7.5 + W6 + S6（27 ÷ 4 = 6.75 → 7.0）

**策略：80% 时间给听力阅读，20% 给写作口语。** 中国考生听力阅读最易涨分，写作口语投入产出比最低。

## 听力建议（无独立 skill）

1. **精听**：剑桥真题，错题那段逐句重听
2. **王陆语料库**：高频词灌耳
3. **错题追踪**：Excel 记错题类型（拼写 / 理解 / 地图定位）
4. **答错不扣分，永不留空**
