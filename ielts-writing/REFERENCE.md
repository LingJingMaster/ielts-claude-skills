# IELTS Writing Reference

这份文件只放参考资料。`SKILL.md` 只保留执行流程。

## 四维评分速查

### Task Response / Task Achievement

| Band | 标准 |
|------|------|
| 7 | 回答了所有部分，立场清晰，论点充分展开，但偶尔过度概括 |
| 6 | 回答了题目但部分论点不够充分，结论可能不清晰 |
| 5 | 只部分回答了题目，论点有限，可能跑题 |

重点检查：
- 是否回答了题目的每个部分
- 立场是否始终一致
- 论点是否有具体展开
- Task 1 是否覆盖关键趋势和数据

### Coherence & Cohesion

| Band | 标准 |
|------|------|
| 7 | 逻辑清晰，衔接自然，段落组织合理，偶尔过度使用连接词 |
| 6 | 有逻辑但衔接有时机械，段落内可能缺少连贯性 |
| 5 | 逻辑不够清晰，段落组织混乱，连接词使用不当 |

重点检查：
- 段落之间是否有递进关系
- 连接词是否自然
- 每段是否只服务一个中心点
- 指代是否清晰

### Lexical Resource

| Band | 标准 |
|------|------|
| 7 | 词汇量足够，能灵活使用不常见词汇，偶尔有搭配错误 |
| 6 | 词汇基本够用，尝试升级词汇但有时不准确 |
| 5 | 词汇有限，经常重复，搭配错误较多 |

重点检查：
- 高频词是否重复
- 是否有有效同义替换
- 搭配是否正确
- 是否有明显拼写错误

### Grammatical Range & Accuracy

| Band | 标准 |
|------|------|
| 7 | 使用多种复杂句型，错误少且不影响理解 |
| 6 | 混合使用简单句和复杂句，有语法错误但不频繁 |
| 5 | 句型有限，错误频繁，部分影响理解 |

重点检查：
- 是否只会简单句
- 是否有主谓一致错误
- 时态是否一致
- 冠词是否频繁出错

## 审题提醒

### Task 2

- 先判断题型：Opinion / Discussion / Advantages-Disadvantages / Problem-Solution / Two-part question
- 标出限定词：`some people`、`in some countries`、`young people` 等
- 多问题题必须全部回应，漏答会直接拖低 TR
- 不要抄题原文
- 立场必须清晰，不能两边都站

### Task 1

- 先识别图表类型：柱状图、折线图、饼图、表格、地图、流程图
- 先看时间范围、单位、比较对象
- 只描述数据，不给个人观点

## 练习出题参考

### Task 2 高频话题

- Education：教育公平、在线学习、考试制度
- Technology：社交媒体、AI、隐私
- Environment：气候变化、可持续发展
- Health：公共健康、心理健康
- Society：城市化、犯罪、贫富差距
- Work：远程工作、自动化、退休年龄

### Task 1 常见类型

- 柱状图 / 折线图 / 饼图 / 表格
- 地图
- 流程图

## 评分校准提醒

- AI 评分普遍偏高 0.5 分，必要时提醒用户实际考试可能更低
- 建议用户用 2-3 个工具交叉验证
- 模板痕迹明显时直接指出，模板文通常锁死 6 分以下
