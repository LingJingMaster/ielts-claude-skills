# IELTS Reading — 参考资料

## 题型分类

| 题型 | 核心能力 | 常见错因 |
|------|---------|---------|
| True/False/Not Given | 逻辑判断 | 混淆 False 和 Not Given |
| Yes/No/Not Given | 观点判断 | 同上，但判作者观点 |
| Matching Headings | 段落概括 | 被细节干扰 |
| Matching Information | 信息定位 | 定位错段落 |
| Matching Features | 人物/理论匹配 | 张冠李戴 |
| Sentence Completion | 信息提取 | 超字数 / 定位错 |
| Summary Completion | 信息提取 | 同上 |
| Multiple Choice | 理解 + 排除 | 没排除干扰项 |
| List of Headings | 段落主旨 | 被首句误导 |
| Table / Flow Chart | 信息提取 | 定位错 |

## TFNG 陷阱

| 陷阱 | 示例 | 正确判断 |
|------|------|---------|
| 原文 A，题目 A+B | 原文说"tea prevented disease"，题目说"tea and beer prevented disease" | 检查 beer 是否也被提到 |
| 比较级 vs 最高级 | 原文"more attractive"，题目"the most attractive" | 不同陈述 |
| 不同原因 | 原文因 X，题目因 Y | FALSE（X≠Y 且原文明确） |
| 限定词 | 原文"some"，题目"all" | FALSE |
| 无时间/地点 | 题目加"in the 18th century" | 检查原文是否提到这个时间 |

## Matching Headings 专项

### 分析结构

```markdown
### 段落 {X} 标题匹配

**段落核心：** {一句话概括}
**首句：** "..."
**尾句：** "..."
**关键词：** {反复出现的主题词}

**正确标题：** {x} — 标题中的"{关键词}"对应段落中的"{对应表述}"
**干扰标题：** {y} — 描述的是 {细节/另一段}，不是本段主旨
```

### 通用策略

- 先读所有标题，划关键词
- 从最容易确定的段落开始（有明显主题句的）
- 段落核心 = 首句 + 尾句的交集，中间是论据
- 首句若是过渡（However / On the other hand）→ 主旨在后面
- 排除法：确定的先填，缩小剩余选项

## 时间管理

| 文章 | 建议时间 | 理由 |
|------|---------|------|
| Passage 1 | 15 分钟 | 最简单，快速拿分 |
| Passage 2 | 20 分钟 | 中等难度 |
| Passage 3 | 25 分钟 | 最难，最多时间 |

超时 → 剩余题目全猜（答错不扣分，猜中 25-33%）。
