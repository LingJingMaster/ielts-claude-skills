## 错题分类

### 听力

- Section 1-2 错多 → 基础词汇 / 拼写
- Section 3-4 错多 → 学术场景 + 语速（正常，优先练）
- 填空错多 → 拼写，用王陆语料库
- 选择错多 → 理解，要精听

### 阅读

- T/F/NG 错多 → 逻辑判断 → `/ielts-reading` 专项
- Matching Headings 错多 → 段落概括弱 → 扫读首尾句
- 填空错多 → 定位弱 → 平行阅读法
- 超时 → 时间管理（15 + 20 + 25 分钟）

## 8 周计划

| 阶段 | 时间 | 重点 |
|------|------|------|
| 摸底 | 第 1 周 | 1 套真题定位 + Simon 写作课 + 口语题库 |
| 建基础 | 第 2 周 | `/ielts-vocab` + 语料库 + 5 个万能故事 |
| 分科强化 | 第 3-4 周 | 按时间分配执行 + 周末模考 |
| 冲刺 | 第 5-6 周 | 60% 给弱项 + 周末模考 |
| 考试模式 | 第 7-8 周 | 隔天全真模考 + 只巩固不学新 |

## 特殊情况处理

| 情况 | 处理 |
|------|------|
| 没模考就来问 | 「先做一套剑桥真题再来。没数据没法诊断。」 |
| 目标不现实（1 月 5→7.5） | 「时间不够。延长备考 or 降到 6.5-7。」 |
| 四科都弱 (<5.5) | 「问题不是技巧是基础。先 1 月补词汇语法，再刷题。」 |
| 考 3+ 次没达标 | 「问题大概率不是学习量，是方法。说说你之前怎么准备的。」 |
| 小分要求（写作≥7） | 调整策略，不能"放弃写作"，要专攻 |
